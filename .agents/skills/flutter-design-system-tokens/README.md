# Flutter Design System Tokens

This skill helps turn Figma exports, token JSON, or style-guide references into a Flutter design system.

Use it when you need to:
- map colors, semantic colors, spacing, radius, typography, and dimensions into Flutter token models
- update theme wiring for the host project
- refresh preview or documentation pages for the design system

The skill is intentionally generic so it can work in different Flutter projects, not just one app.

Expected inputs:
- Figma export JSON or token JSON
- design-system reference images or screenshots
- the existing Flutter project's token/theme structure

Expected outputs:
- updated token source files
- updated Flutter model and theme code
- updated preview/demo pages when available
